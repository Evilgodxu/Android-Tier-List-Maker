# 3.0.0 更新日志

## 新增功能

### 视频生成

- 编辑过程中无感录制图片放置、命名、小图标添加动作
- 为每张图片独立绑定/上传解说音频（MP3 / WAV / M4A）
- 图片间隔来源可选：手动固定时长，或按解说音频时长自动计算
- 自定义三种动作类型播放顺序：图片放置、图片命名、小图标添加
- 两种图片播放顺序：按图片顺序、按层级顺序
- 使用 MediaCodec / MediaMuxer 合成 MP4（H.264 + AAC），默认 1920×1080
- 编码失败时自动尝试减半分辨率重试
- 编排配置可随榜单预设一起保存与复用

### 其他

- 支持 10 种语言运行时切换
- 深色/浅色主题跟随系统
- 待分级图片区与层级列表支持自适应布局
