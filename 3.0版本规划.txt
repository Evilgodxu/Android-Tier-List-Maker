一、版本定位

梯度大师 2.0 提供了静态角色强度榜编辑能力，并支持预设的保存、导出、导入与管理。3.0 版本将编辑器升级为 “榜单编辑过程视频生成器”，核心目标：

· 用户在编辑榜单时，系统自动无感录制所有操作（图片放置、小图标添加、图片命名）。

· 生成视频时，用户可以完全自定义三种动作的播放顺序、编排粒度及所有时间间隔，形成差异化节奏的视频。

· 支持动画侧载与位图生成侧载，增强视频表现力。

· 增强预设能力：将编排配置保存为预设，支持快速复用到更新后的榜单数据。

· 层级顺序即播放顺序：用户通过拖拽调整层级列表中的角色顺序，即可控制视频中图片放置的先后次序。

· 音频控制间隔：为每张图片独立录制或上传解说音频，系统根据音频时长自动调整图片间的播放间隔，彻底解决手动配时长的难题，无需任何 AI 服务。

二、核心功能

2.1 无感录制模块

用户正常编辑榜单，系统自动记录以下动作（无时间戳）：

动作类型 记录内容

图片放置 哪张图片被放入了哪个层级（按层级列表顺序）

小图标添加 为哪张图片添加了第几个小图标（最多3个）

图片命名 为哪张图片设置了什么文字名称

录制不限制操作顺序，不保留任何时间信息。层级顺序即播放顺序，用户拖拽调整层级列表即可改变视频中图片放置的先后次序。

2.2 音频解说与间隔控制（核心新增功能）

2.2.1 音频上传/录制

· 为每张图片（每个角色）独立添加一段解说音频（支持 MP3 / WAV / M4A 格式）。

· 用户可以：

· 上传本地音频文件（例如用手机录制的解说）

· 直接在编辑器中录制音频（调用麦克风，支持简单裁剪与重录）

· 该音频不会影响视频中原有的动作音效，可作为独立音轨叠加。

2.2.2 自动间隔控制

· 系统自动读取每段音频的实际时长（秒）。

· 在视频生成时，该时长将作为从当前图片的“所有动作完成”到下一张图片开始放置之间的等待时间。

· 例如：图A的解说音频时长为 2.3 秒，则图A的所有动作（如放置+命名+图标）完成后，视频会停留 2.3 秒（同时播放图A的解说音频），再开始图B的放置动作。

2.2.3 间隔配置选项

用户可在“视频生成设置”中统一控制间隔策略：

配置项 说明

图片间隔来源 • 手动固定时长（与之前相同） • 音频时长（使用上传的解说音频长度作为间隔）

音频叠加方式 • 作为背景解说独立播放 • 同时保留原动作音效 • 仅保留音频，静音动作音效

额外偏移 在音频时长基础上统一增加一个固定偏移秒数（例如 +0.3 秒），用于留白或转场

2.2.4 应用场景

· 用户无需猜测每句话该留多少秒，直接录制或上传配音，视频节奏自动匹配。

· 支持多人分工：写稿者、录音者、编辑器操作者可分离。

· 完全不依赖任何 AI 服务，所有音频由用户自己提供，隐私与可控性高。

2.3 播放顺序与编排粒度配置

进入 “视频生成设置” 面板，可进行两项核心配置：

2.3.1 动作类型顺序

自由拖拽排列三种动作类型的相对顺序。预设模板：

· 顺序 A：图片放置 → 图片命名 → 小图标添加

· 顺序 B：图片命名 → 图片放置 → 小图标添加

· 顺序 C：小图标添加 → 图片命名 → 图片放置

2.3.2 编排粒度

决定动作在不同图片之间如何交错执行：

粒度模式 说明

按图片串行 针对第一张图片完整执行所有动作类型（按用户设定的顺序），完成后才切换到第二张图片

按类型批量 先对所有图片执行第一种动作类型，再对所有图片执行第二种动作类型

混合模式 自定义每个动作类型是“逐图片轮询”还是“批量执行”

2.4 间隔与动效配置（完整参数）

配置项 说明 可选值/方式

图片放置间隔 图片之间的等待时间，可选择： • 手动固定时长 • 音频时长（自动匹配每张图片的解说音频长度） 0.1~10.0秒 或 音频时长

小图标显示间隔 同一图片内多个图标之间的播放间隔 0.1~5.0秒

命名显示间隔 一次性显示或逐字显示（可设每字间隔） -

跨类型停顿 切换到下一类型动作前的全局等待时间 0~5.0秒

跨图片停顿（串行模式） 完成一张图片的全部动作后，切换到下一张图片前的额外停顿（不含音频间隔） 0~5.0秒

2.5 完整示例（包含音频间隔）

录制数据：2 张图片，层级顺序为图A（T0）、图B（T1）。

用户设置：动作顺序 = 放置 → 命名 → 图标；编排粒度 = 按图片串行；

命名间隔 = 0.5 秒；图标间隔 = 0.3 秒；

图片间隔来源 = 音频时长；图A解说音频时长 2.3 秒，图B解说音频时长 1.8 秒；无额外偏移。

时间线：

· 0.0s 放置图A

· 0.5s 显示图A名称

· 0.8s 添加图A图标

· 2.3s （播放图A解说音频，持续 2.3 秒）

· 2.3s 放置图B

· 2.8s 显示图B名称

· 3.1s 添加图B图标

· 4.9s （播放图B解说音频，持续 1.8 秒）

解说音频与画面动作同步播放，时长自动控制图片切换节奏。

2.6 视频生成引擎

· 输出格式：MP4（H.264），默认 1920×1080

· 合成逻辑：根据动作顺序、编排粒度、间隔配置（含音频时长）逐帧渲染，并混入解说音频与可选背景音乐。

· 音频轨道：解说音轨 + 背景音乐（可独立控制音量）。

2.7 侧载资源支持

· 动画侧载：导入 Lottie/GIF/帧序列，绑定到动作（如命名时弹出光效）。

· 位图生成侧载：批量导入位图，按播放顺序自动匹配到角色。

2.8 编排预设复用机制（基于2.0预设体系扩展）

3.0 将全部编排配置（动作顺序、编排粒度、间隔参数、音频绑定、动效绑定、文本样式、背景音轨）保存为编排预设，与具体榜单数据解耦。

典型使用场景：

1. 首次制作：为每张图片录制/上传解说音频，配置好顺序、粒度、间隔规则，保存为预设。

2. 游戏更新：新增角色后，用户拖拽调整层级顺序，并为新角色录制音频。

3. 一键复用：加载之前保存的预设，系统自动根据当前层级顺序和音频时长生成视频，新角色自然第一个展示，节奏自动匹配。

预设管理：支持保存、导出/导入、重命名、分组、收藏。

三、用户操作流程（完整示例）

1. 编辑榜单：添加角色图片、命名、小图标，拖拽层级顺序决定强度排名。

2. 录制/上传解说音频：

· 为每个角色点击“录制音频”按钮，用麦克风直接解说（或上传本地文件）。

· 可预览音频，不满意可重录。

3. 配置视频参数：

· 选择动作顺序、编排粒度

· 设置间隔来源为“音频时长”

· 调整其他间隔参数（如命名间隔、跨图片停顿）

· （可选）绑定侧载动画、添加背景音乐

4. 预览：系统快速生成低分辨率预览，检查画面与音频是否同步。

5. 导出视频：生成最终 MP4 文件（包含解说音频 + 背景音乐）。

6. 后续更新：游戏新增角色后，拖拽调整层级顺序，为新角色录制音频 → 加载预设 → 导出新视频。无需重新配置任何间隔。
